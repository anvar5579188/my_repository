ls   - показывает все файлы и папки внутри текущей директории
ls test   - показывает все файлы и папки внутри папки test
ls -a   - показывает также все скрытые файлы и папки
ls -l   - показывает более подробную информацию о файлах и папках
e
cd   - переходит в домашнюю папку
cd ~   - переходит в домашнюю папку
cd test   - переходит в папку test, если такая папка существует в текущей директории
cd .   - переходит в текущую папку (остается в ней же)
cd ..   - переходит в родительскую папку
cd ../..   - переходит на 2 папки назад

mkdir test   - создает папку test в текущей директории
mkdir test/test2    - создает папку test2 в папке test, если папка test существует
mkdir test1 test2   - cоздает папки test1 и test2 в текущей директории

touch test.txt   - создает файл test.txt в текущей директории
touch test{1,2}.txt   - создает файлы test1.txt и test2.txt

rm test   - удаляет файл test (если это не папка)
rm -rf test   - удаляет файл или папку со всем содержимым (удаляет безвозвратно)

pwd   - показывает абсолютный (начиная с корневой папки /) путь до текущей директории
(print working directory)

cat test.txt   - показывает содержимое файла
nano test.txt    - открывает файл test.txt в редакторе кода nano
code test    - открывает папку или файл test в редакторе кода VSCode

sudo   - права суперюзера (super user do)

(apt - системный установщик пакетов)
(snap - установщик пакетов (чаще всего каких-либо приложений))
